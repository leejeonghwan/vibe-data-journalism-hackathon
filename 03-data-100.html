공공데이터 100선 — 던져 보기 좋은 데이터

data.go.kr에서 "대충 던져 보고 단서를 찾기" 좋은 데이터 100개.

각 항목은 데이터셋 + 던질 질문 + 기대 단서로 되어 있다. 그대로 코워크에 던지면 탐색이 시작된다.

2강과 함께 쓴다
이 100선은 기초취재 방법론의 연료다. 여기서 데이터를 하나 골라 "대충 던지기 → 뽑기 → 어? 포착 → 검증 → 좁히기" 루프를 돌린다. 한 항목이 곧 한 번의 실습이 된다.

왜 data.go.kr 하나면 충분한가

공공데이터포털은 16개 분류(공공행정·보건의료·사회복지·재난안전·환경기상 등)에 걸쳐 수만 건의 데이터를 한곳에 모아 둔다. 특히 두 가지가 초보 훈련에 결정적이다.

  • 파일 파일데이터(엑셀·CSV)는 로그인·키 없이 즉시 다운로드. 받아서 코워크에 던지기만 하면 된다. 초보 1순위.
  • API 오픈API는 회원가입·키 신청이 필요하지만, 시계열·전국 단위 대량 수집에 강하다. 익숙해지면 넘어간다.

여기에 LOCALDATA(지방행정 인허가 195종: 일반음식점·통신판매·미용·담배소매 등)와 전국 표준데이터(어린이집·주차장·도서관 등 전국을 같은 양식으로)가 더해져, "전국을 한 양식으로 비교"하는 기사가 쉬워진다.

쓰기 전 한 가지
데이터셋의 정확한 제목·연도는 수시로 바뀐다. 표의 이름은 검색 키워드로 쓰고, data.go.kr 상단 검색창에 넣어 최신본을 찾는다. "파일데이터" 탭부터 보면 키 없이 받을 수 있는 것만 추려진다.

각 항목 읽는 법

칸뜻
데이터셋data.go.kr에서 검색할 이름 + 파일/API 표시
던질 질문코워크에 그대로 던질 탐색 시작 질문
기대 단서잘 풀리면 어떤 "어?"가 나올 수 있는지 — 정답이 아니라 방향
공통 첫 문장
이 폴더의 [파일]을 원본은 두고 사본으로 작업해라. 먼저 무엇이 들었는지 훑고, 기자가 눈여겨볼 이상한 지점을 근거와 함께 짚어 달라.
난이도 기준
파일은 초보·주말 실습용, API는 시계열·전수 수집이 필요한 심화용. 첫 해커톤은 파일데이터 위주로 배정하면 데이터를 받지 못해 시간을 날리는 일이 없다.
